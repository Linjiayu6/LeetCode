# 位运算
## 背景
- 二进制存在设备中, 有0,1两种状态;
- 每一种语言都会转化成机器语言去执行;

## 计算机计算原理
### 加法
```
a, b = 1, 2
c = a + b

a, b = 001 + 010
c = 001 + 010 = 011
```

### 乘法
```
a, b = 3, 2
c = a * b

a, b = 011, 010
c = 011 * 2 = 110
```

```
a, b = 3, 4
c = a * b

a, b = 011, 100
c = 011 * 4 = 1100
```

- 当满足 2^N 相当于二进制向左移动N位 a << N
- 3 * 2 = 3 << 1  3 * 4 = 3 << 2

```
a, b = 15, 13
c = a * b

b = (4 + 8 + 1)
c = 15 * 4 + 15 * 8 + 15 * 1

# 二进制相当于
c = 15 << 2 + 15 << 3 + 15 << 0
```

### 运算
- [左 << 右 >> 移动] 001 << 1 = 010  111 >> 1 = 011
- [或 | 与 & | 取反 ~]
- [异或 ^] 相同为0 不同为1
